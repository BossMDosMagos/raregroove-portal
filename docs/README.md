Docs - RareGroove
=================

This folder groups all project documentation by purpose. Use the index in docs/guias/INDICE-DOCUMENTACAO.md to navigate quickly.

Structure
---------
- docs/guias: main index and general documentation entry points
- docs/guias-rapidos: quick start guides and run-now instructions
- docs/guias-completos: full system guides and deep dives
- docs/sql: all SQL scripts (setup, fixes, queries)
- docs/processos: troubleshooting, procedures, internal notes
- docs/checklists: production and verification checklists

Conventions
-----------
- Keep new SQL scripts in docs/sql and reference them with relative links
- Quick guides should link to full guides when available
- When moving a doc, update links in docs/guias/INDICE-DOCUMENTACAO.md

Useful Entry Points
-------------------
- docs/guias/INDICE-DOCUMENTACAO.md
- docs/guias-rapidos/EXECUTAR-AGORA.md
